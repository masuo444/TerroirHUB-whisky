# Terroir HUB WHISKY — 制作ルール

## 1. 情報の正確性（最重要）

### 嘘をつかない原則
- **蒸留所が公式に出している情報だけを使う**
- 情報がない項目は「表示しない」。推測で埋めない
- AIが勝手に文章を生成・補完しない

### 引用ルール
- 蒸留所サイト・公式SNSからの文章引用には**必ず出典を明記**する
  ```html
  <div style="font-size:10px;color:var(--text-muted);margin-top:10px;">
    出典：<a href="https://example.com" target="_blank" rel="noopener">蒸留所名 公式サイト</a>
  </div>
  ```
- マスターブレンダー・蒸留所長の個人名での発言は、**その人物が実際にその発言をしたと確認できる場合のみ**帰属する
- 確認できない場合は蒸留所の一般的な説明として記述する

### データソースの優先順位
1. 蒸留所公式サイト（最優先）
2. ラベル法定表記（アルコール度数・原材料・年数表記等）
3. 公式コンペティション結果（WWA / ISC / TWSC / IWC）
4. 国税庁オープンデータ（蒸留所名・住所・創業年）
5. 日本洋酒酒造組合の公式情報
6. 蒸留所のSNS公式アカウント

### 使ってはいけないソース
- 個人ブログ・口コミサイトの主観的評価
- AIによる推定・生成テキスト（ファクトチェック不可のもの）
- 出典不明の二次情報

### ウイスキー固有の情報正確性ルール
- **年数表記（Age Statement）は公式ラベル通りに記載** — NASの商品に年数を付けない
- **樽タイプは公式発表情報のみ** — 味の推測から樽種を特定しない
- **2024年施行のジャパニーズウイスキー表示基準を遵守** — 日本洋酒酒造組合の自主基準に基づく表示
  - 「ジャパニーズウイスキー」表示の条件：原料・糖化・発酵・蒸留・熟成すべて日本国内
  - 基準を満たさない製品に「ジャパニーズウイスキー」の表記を使わない
- **カスクストレングスの度数は正確に** — 小数点以下まで公式通りに記載

## 2. 銘柄カードの情報設計

### 必須項目（ラベル・公式サイトから取得可能）
- 銘柄名
- 種類（シングルモルト・ブレンデッド・グレーン等）
- アルコール度数
- 主要原料
- 説明文（公式サイトの記述ベース）

### 条件付き表示項目（データがある場合のみ表示）
- **年数表記**：12年 / 18年 / 25年 / NAS
  - 蒸留所公式表記がある場合のみ表示
  - NASの場合は「ノンエイジ」と表示
- **樽タイプ**：バーボン樽 / シェリー樽 / ミズナラ樽 / ワイン樽 etc.
  - 公式に記載がある場合のみ
- **カスクストレングス**：度数が50%以上の場合に「カスクストレングス」タグを表示
- **受賞歴**：年・コンペ名・賞名
  - 公式発表で確認できるもののみ
  - 受賞歴がない銘柄にはセクション自体を表示しない

### 画像
- 正方形（aspect-ratio: 1/1）のプレースホルダーを標準装備
- 蒸留所から使用許可を得た写真 → `<img class="brand-img">` に差替
- 未許可の蒸留所 → プレースホルダー表示（`brand-img-placeholder`）
- Unsplash等の汎用画像は**蒸留所や銘柄の固有ページには使わない**（誤解を招く）
- ポータルトップの一般的なセクション（地域紹介等）ではUnsplash使用可

## 3. ジャパニーズウイスキーの表示基準

### 2024年表示基準（日本洋酒酒造組合自主基準）
- **「ジャパニーズウイスキー」と表示可能な条件**：
  - 原材料の穀物が日本国内で糖化・発酵・蒸留
  - 日本国内で3年以上樽熟成
  - 日本国内で瓶詰め
- 基準未対応の蒸留所・商品には「ジャパニーズウイスキー」ラベルを付与しない
- 「ジャパニーズスピリッツ」「ウイスキー」と区別して表示

### クラフト蒸留所の扱い
- 新興のクラフト蒸留所も大手と同格の第一級カテゴリとして表示
- 蒸留開始年と最初の商品リリース年を区別して記載
- 「準備中」「蒸留開始」「初リリース済み」のステータスを明記

- **ディスプレイ**：Zen Old Mincho 700（蒸留所名・ヒーロー見出し）

- **本文**：Noto Sans JP 400, 17px, line-height 1.85
- **UI・数値**：DM Sans
- **最小フォントサイズ**：14px（全世代対応）

### レイアウト
- 銘柄カード：3列グリッド、1px gap、ボーダー区切り
- カードホバー：上部にディープアンバーグラデーションライン出現
- border-radius：6〜12px
- ボックスシャドウは控えめに（ボーダー主体）

## 5. 多言語対応

### 対応言語
| 言語 | パス | 優先度 |
|------|------|--------|
| 日本語 | `/whisky/{pref}/{id}.html` | 最優先 |
| 英語 | `/whisky/en/{pref}/{id}.html` | 高（世界市場） |
| フランス語 | `/whisky/fr/{pref}/{id}.html` | 高（ウイスキー愛好市場） |

## 6. SEO・AIO戦略

### 構造化データ（必須）
- 全蒸留所：LocalBusiness + Product schema
- 教科書ページ：FAQPage schema
- 飲み方ガイド：HowTo schema
- パンくずリスト：BreadcrumbList schema
- サイト検索：SearchAction schema

### AIO（AI Overview）対策
- 教科書ページは「問い→答え」形式で構成
- 1段落200文字以内で結論を先に述べる
- 英語版を必ず用意（英語圏のAI Overviewに引用されるため）
