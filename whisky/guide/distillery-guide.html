CHAPTER 06

蒸留所巡りガイド

ポットスチルの熱気、樽貯蔵庫の香り、蒸留所でしか飲めない限定ボトル。画面では味わえない体験が、蒸留所にはあります。

見学できる主要蒸留所

山崎蒸溜所（大阪府島本町）

1923年創業、日本最古のモルトウイスキー蒸留所。ポットスチルの見学からテイスティングまで充実のツアー。併設のウイスキーライブラリーも必見。

予約: 公式サイトから必須（数カ月前推奨） / 料金: 有料 / アクセス: JR山崎駅 徒歩10分

白州蒸溜所（山梨県北杜市）

南アルプスの麓、標高700mの森の中に佇む蒸留所。自然遊歩道を散策しながらウイスキーの世界を体験できます。

予約: 公式サイトから必須 / 料金: 有料 / アクセス: JR小淵沢駅 タクシー10分

余市蒸溜所（北海道余市町）

竹鶴政孝が理想の地として選んだ蒸留所。世界でも希少な石炭直火蒸留を今も続けています。自由見学が可能で敷居が低いのも魅力。

予約: 自由見学は不要（ガイドツアーは要予約） / 料金: 無料 / アクセス: JR余市駅 徒歩3分

宮城峡蒸溜所（宮城県仙台市）

広瀬川と新川の合流地点に位置。余市とは対照的な華やかで軽やかな原酒を生む、ニッカのもう一つの顔。

予約: 要予約 / 料金: 無料 / アクセス: JR作並駅 シャトルバス

秩父蒸溜所（埼玉県秩父市）

イチローズモルトで世界を席巻するベンチャーウイスキー。小規模ながら妥協のないウイスキー造りを間近で見学できます。

予約: 完全予約制（少人数） / 料金: 有料 / アクセス: 秩父鉄道 皆野駅 タクシー

おすすめルート

関西ウイスキートレイル（2日間）

1日目: 山崎蒸溜所 → 京都泊。2日目: 白州蒸溜所（特急あずさで移動）。日本ウイスキーの原点を巡る王道コース。

北海道クラフトジャーニー（3日間）

1日目: 余市蒸溜所 → 小樽泊。2日目: 二世古蒸溜所 → ニセコ泊。3日目: 厚岸蒸溜所。北の大地のウイスキーを堪能。

関東日帰りコース

秩父蒸溜所（午前） → 秩父市内でランチ＆地酒 → 東京へ帰還。クラフトウイスキーの聖地を気軽に訪問。

見学のマナーと持ち物

蒸留所内は歩きやすい靴で。製造エリアでは係員の指示に従い、火気厳禁。試飲がある場合は必ず公共交通機関を利用してください。カメラは撮影可能エリアのみ。冬場の北海道や山間部は防寒対策を忘れずに。
